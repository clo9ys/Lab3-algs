# Алгоритмический мини‑пакет структур данных и сортировок

**Данная программа включает в себя несколько алгоритмов сортировки, функции вычисления факториала и чисел Фибоначчи, а также структуру данных стек (реализация на list).**

## Требования
- Python 3.10 или выше  
- Установленные пакеты: `typer`, `pytest`, `pytest-cov`

## Структура проекта
- `src/*.py` – реализации сортировок, стек, генераторы массивов и вспомогательные функции
- `cli.py` – Typer‑CLI с командами
- `main.py` – точка входа интерактивного меню
- `tests/*.py` – unit‑тесты для основных функций

## Запуск программы

Программа поддерживает:
- прямой CLI‑запуск отдельных команд через Typer;
- интерактивное меню для пошагового выбора действий.

### CLI‑режим

```
python -m cli fibo 10
python -m cli fibo 10 -r           # рекурсивный вариант
python -m cli factorial 5
python -m cli sort quick "3 1 2"
python -m cli stack                # режим работы со стеком
python -m cli benchmark            # запуск бенчмарков
```

### Интерактивное меню

```
python -m main
```

**После запуска выбирайте номер нужного пункта.**

## Тестирование

```
python -m pytest
```

## Обработка ошибок
- `IndexError` - попытка сделать pop/peek/min в пустом стеке
- `ValueError` - передан отрицательный элемент в функцию поиска факториала или числа фибоначчи
